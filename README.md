# Gerenciador de Orçamento - Backend

Projeto desenvolvido em Spring Boot para gerenciamento de Orçamentos e Medições.

## Tecnologias Utilizadas

- Java 21
- Spring Boot 3
- Spring Data JPA
- PostgreSQL
- Swagger (OpenAPI)
- Lombok

---

## Como Executar o Projeto

### 1. Pré-requisitos

- Java 21 instalado
- Maven instalado
- PostgreSQL instalado
